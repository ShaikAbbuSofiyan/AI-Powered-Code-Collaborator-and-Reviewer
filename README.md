# 🚀 AI-Powered Code Review & Collaboration Platform

A full-stack **MERN (MongoDB, Express, React, Node.js)** application that enables developers to **write, analyze, and collaborate on code** with the help of **AI-driven insights**.

This platform combines features of **code editors, collaboration tools, and intelligent code review systems** into a single unified workspace.

---

# 🧠 Key Features

## 🔐 Authentication & User Management

* Secure login & signup using JWT
* Password hashing with bcrypt
* Protected routes and user sessions

---

## 📂 Project & Code Management

* Create and manage coding projects
* Store multiple files per project
* Integrated **Monaco Editor** for a VS Code-like experience

---

## 🤖 AI Code Review (Core Feature)

* Analyze code for:

  * Bugs and errors
  * Performance issues
  * Optimization opportunities
* Suggest improvements and best practices
* Provides structured feedback using LLM APIs

---

## 💬 Real-Time Collaboration

* Live chat and discussions using WebSockets
* Comment on code and collaborate with team members
* Mention users and track conversations

---

## 🐞 Issue Tracking System

* Create, assign, and manage issues
* Track progress with statuses (Open, In Progress, Done)

---

## 📊 AI-Based Code Quality Analysis

* Evaluates:

  * Readability
  * Efficiency
  * Maintainability
* Helps developers improve coding standards

---

# 🧱 System Architecture

* **Client Layer:** React-based UI with code editor and collaboration tools
* **Backend Layer:** REST APIs for authentication, projects, and AI processing
* **AI Layer:** LLM integration for code analysis and suggestions
* **Data Layer:** MongoDB for storing users, projects, and issues
* **Realtime Layer:** WebSocket-based communication for collaboration

# ⚡ Getting Started

## 1️⃣ Clone the repository

```bash
git clone https://github.com/ShaikAbbuSofiyan/AI-Powered-Code-Collaborator-and-Reviewer.git
cd AI-Powered-Code-Collaborator-and-Reviewer
```

---

## 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
OPENAI_API_KEY=your_api_key
```

Run backend:

```bash
npm run dev
```

---

## 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```
